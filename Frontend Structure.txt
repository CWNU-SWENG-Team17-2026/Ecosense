라우팅 (App.jsx)
  /              → StartPage
  /login         → LoginPage
  /register      → RegisterPage
  /main          → MainPage
  /outdoor       → OutdoorPage (Layout)
  /indoor        → IndoorPage (Layout)
  /sleep         → SleepPage (Layout)
  /reports       → ReportsPage (Layout)
  /settings      → SettingsPage (Layout)

환경 변수
  frontend/.env.local
    VITE_API_BASE_URL=http://127.0.0.1:8000/api
  ※ localhost와 127.0.0.1은 쿠키 도메인이 달라서 반드시 통일할 것

주요 데이터 흐름
1. 게스트 수면 측정
   useAudio → useNoiseStore + indexedDB(spikes, sessions)
   SessionHistory → getSpikesBySession (로컬)

2. 로그인 사용자
   AuthBootstrap → /auth/me
   useSync → upload/download (설정에서 syncAll)
   SessionHistory → /api/sessions, /api/sessions/{id}/spikes

3. 실외 데이터
   OutdoorPage 마운트 시 자동 loadOutdoorData
   GPS → "lat,lon" → loadOutdoorData 자동 호출
