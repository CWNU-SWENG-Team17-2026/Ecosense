라우팅 (App.jsx)
  /              → StartPage
  /login         → LoginPage
  /register      → RegisterPage
  /survey        → SurveyPage
  /main          → MainPage
  /outdoor       → OutdoorPage (Layout)
  /indoor        → IndoorPage (Layout)
  /sleep         → SleepPage (Layout)
  /reports       → ReportsPage (Layout)
  /settings      → SettingsPage (Layout)

환경 변수
  VITE_API_BASE_URL=http://127.0.0.1:8000/api   (로컬)
  VITE_API_BASE_URL=https://xxx.onrender.com/api (Vercel 배포)
  ※ Render URL만 넣어도 api.ts가 /api suffix 자동 보정

배포
  Frontend: Vercel (ecosense-sooty.vercel.app 등)
  Backend:  Render Web Service
  DB:       Render PostgreSQL

주요 데이터 흐름
1. 회원가입: register → verify → login → (선택) survey
2. 실외: OutdoorPage → getOutdoorData → KMA+에어코리아 (백엔드 프록시)
3. 수면: useAudio → indexedDB + useNoiseStore (게스트/로그인 공통 로컬)
4. 동기화: useSync → /api/sync/upload|download (로그인 시)
